# Social Network — Personal Development Platform

Projet personnel full-stack visant à concevoir une plateforme sociale centrée sur le développement personnel et les objectifs individuels.

Contrairement aux réseaux sociaux traditionnels orientés contenu visuel ou divertissement, cette plateforme plaçait les **objectifs, projets et progression personnelle** au cœur de l’expérience utilisateur.

L’ambition était de réunir une communauté autour de projets communs et de créer un environnement de saine émulation et de responsabilisation.

---

## Vision Produit

- Profils centrés sur les objectifs personnels
- Mise en avant des projets en cours plutôt que du contenu éphémère
- Interactions favorisant l’encouragement et le suivi de progression
- Dynamique communautaire orientée engagement constructif

---

## Stack Technique

**Frontend**
- React Native (Expo)
- TypeScript
- Context API
- Architecture modulaire scalable

**Backend**
- Node.js
- Express (API REST)
- PostgreSQL

---

## Architecture

Le projet a été conçu et développé intégralement en autonomie.

### Front-end
- Architecture modulaire organisée par domaine fonctionnel
- Séparation claire entre UI, logique métier et services API
- Gestion d’état globale via Context API
- Centralisation des appels API
- Optimisation des re-renders et gestion asynchrone avec async/await

### Backend
- API REST structurée par ressources
- Middleware d’authentification JWT
- Validation des requêtes
- Gestion sécurisée des sessions
- Organisation modulaire des routes et contrôleurs

### Base de données
- Modélisation relationnelle PostgreSQL
- Gestion des utilisateurs, publications, commentaires et interactions
- Relations structurées pour assurer cohérence et évolutivité

---

## Fonctionnalités Implémentées

- Création de compte et authentification sécurisée
- Gestion de profils utilisateurs
- Création et suivi d’objectifs personnels
- Publications liées aux projets
- Interactions sociales (likes, commentaires)
- Gestion des permissions utilisateurs
- Upload et affichage de médias

---

## Objectifs Techniques

Ce projet avait pour but de :

- Maîtriser la conception d’une architecture full-stack complète
- Structurer une application mobile scalable
- Mettre en place un système d’authentification sécurisé
- Concevoir une API REST robuste
- Travailler la modélisation de données relationnelle
- Approfondir la gestion des flux asynchrones

---

## Statut

Projet personnel expérimental ayant permis une exploration approfondie de l’architecture applicative mobile et full-stack.
